Rust Tips: Box Rc Arc Cell Refcell Mutex

March 08, 2021

Difference Between Box, Rc, Arc, Cell (and its various variants.. RefCell, Mutex, RwLock) in Rust: Box is for single ownership. A great use case is to use this when we want to store primitive types (stored on stack) on heap. Rc is for multiple ownership. Arc is for multiple ownership, but threadsafe. RefCell is for “interior mutability”; that is, when you need to mutate something behind a &T. Cell is for “interior mutability” for Copy types; that is, when you need to mutate something behind a &T.
